Recommander
V 1.2

 

 

Installation :

Décompressez le fichier zip dans votre répertoire de modules (/modules/) puis ouvrez le fichier nommé "include_var.php" et modifiez les lignes 7, 9, 11, 13 et 21.
Merci de laisser les crédits.
Uploadez maintenant le répertoire recommander (de votre dossier modules) avec votre logiciel de FTP favoris.
Vous pouvez maintenant aller installer le module sur votre site.

 

Informations :

Avec ce script, les visiteurs et/ou membres de votre site pourront conseiller le site à 5 ami(e)s à la fois et ils pourront même laisse run commentaire. Vous pouvez même recevoir une copie du mail qu'a envoyé l'internaute !
Le script vérifie si l'internaute a bien entré son nom, son adresse e-mail, et au moins le nom et l'adresse e-mail de l'ami(e) N°1.
Le script regarde aussi si l'internaute a entré un commentaire sinon il lui attribue la valeur "nom de l'internaute n'a pas entré de commentaire."
Attention ! Pour que ce script marche, votre hébergeur doit avoir activé la fonction mail() de PHP.

 

Informations relatives à la version de Xoops utilisée :

Ce script a été testé sous Xoops V 2.0.6 mais devrait marcher sous d'autres version grâce à la simplicité de son code (Codeurs : vous n'aurez qu'à regarder ;-) )
